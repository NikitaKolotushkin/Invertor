# Inventory-Control
Школьный проект

# Установка
### Windows:

+ ```git clone https://github.com/NikitaKolotushkin/Inventory-Control.git```
+ ```cd Inventory-Control```
+ ```pip install -r requirements.txt```
+ ```python main.py```
+ Перейти в браузере на 127.0.0.1:5000


### Linux:

+ ```git clone https://github.com/NikitaKolotushkin/Inventory-Control.git```
+ ```cd Inventory-Control```
+ ```pip install -r requirements.txt```
+ ```python3 main.py```
+ Перейти в браузере на 127.0.0.1:5000
